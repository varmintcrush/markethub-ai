# MarketHub AI

## Technologies

- Next.js
- TypeScript
- Express.js
- PostgreSQL
- Prisma
- Redis
- FastAPI
- ChromaDB
- Cloudinary
- Socket.IO
- Docker

## Features

- Authentication
- Role Based Access
- Product Management
- Category Management
- Shopping Cart
- Orders
- AI Semantic Search
- Real-time Chat

## Installation

Frontend:

npm install
npm run dev

Backend:

npm install
npm run dev

AI Service:

pip install -r requirements.txt
uvicorn main:app --reload
uvicorn main:app --reload
